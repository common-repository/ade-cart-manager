== Description ==

Ade Cart Manager is a powerful plugin that helps you recover lost sales by targeting abandoned carts. With its user-friendly interface and advanced features, it simplifies the process of recovering abandoned carts and enables you to convert them into successful sales.

Using Ade Cart Manager, you can create custom email campaigns to reach out to customers who have left items in their cart without completing the purchase. The plugin also allows you to set up automated reminders and incentives such as discount coupons to encourage customers to complete their purchase.

With Ade Cart Manager, you can track the abandoned carts in real-time and receive notifications whenever a cart is abandoned. The plugin also provides detailed reports and analytics, giving you insights into cart recovery rates and customer behavior.

In summary, Ade Cart Manager is an indispensable tool for any eCommerce store owner looking to recover lost sales and boost revenue. Try it today and take your business to the next level!

== Installation ==

1. Upload `ade-cart-manager` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. And everything works perfectly within woocomerce
